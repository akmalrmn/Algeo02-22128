## Usage
How to run:
- Backend
     1. pip install flask
     2. pip install fastapi
     3. pip install uvicorn
     4. pip install python-multipart
     5. Go to main.py file and then run Uvicorn main:app -refresh

- Frontend
     1. install Node.js
     2. npm install
     3. Go to react-web folder and then run npm run dev
